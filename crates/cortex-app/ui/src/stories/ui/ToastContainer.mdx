# ToastContainer

A zero-prop container that reads `toasts` from `useToastStore` and stacks them in the top-right corner of the viewport (`fixed top-4 right-4 z-[9999]`). Each toast is wrapped in a `pointer-events-auto` div so dismissal works even though the container itself uses `pointer-events-none`.

Source: [`src/components/ui/ToastContainer.tsx`](../../../components/ui/ToastContainer.tsx)

Mount `<ToastContainer />` once in `App.tsx` — it is always present regardless of which workspace or panel is active.

## Behaviour notes

- **Auto-dismiss**: `addToast` schedules a `setTimeout` for `duration` ms (default 4000). Stories use direct `setState` so duration timers are not active.
- **Stacking order**: toasts are appended; newest appears at the bottom of the stack.
- **Z-index**: `z-[9999]` sits above all panels, modals, and dropdowns.
