# MethodBadge

A compact, colour-coded badge that identifies the HTTP method of a request. Each method maps to a dedicated design-token colour pair (`text-method-<m>` / `bg-method-<m>/15`) so badges remain legible across all 13 Cortex themes.

## Props

| Prop        | Type     | Default | Description                                      |
| ----------- | -------- | ------- | ------------------------------------------------ |
| `method`    | `string` | —       | HTTP method string (case-insensitive match)      |
| `className` | `string` | `''`    | Additional Tailwind classes for layout overrides |

The component normalises `method` to lowercase before matching, so `'GET'`, `'get'`, and `'Get'` all render identically. Unknown methods fall back to a muted neutral style.
