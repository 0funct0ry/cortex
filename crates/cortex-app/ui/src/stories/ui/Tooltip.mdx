# Tooltip

A lightweight hover tooltip that renders a small floating label adjacent to its trigger element. Visibility is controlled by `onMouseEnter` / `onMouseLeave` with a configurable delay, and placement is determined by the `position` + `align` props.

Source: [`src/components/ui/Tooltip.tsx`](../../../components/ui/Tooltip.tsx)

---

## Props

| Prop       | Type                                     | Default    | Description                                                 |
| ---------- | ---------------------------------------- | ---------- | ----------------------------------------------------------- |
| `content`  | `string`                                 | —          | Text displayed inside the tooltip                           |
| `children` | `ReactNode`                              | —          | The element that triggers the tooltip                       |
| `delay`    | `number`                                 | `300`      | Milliseconds before the tooltip appears after hover         |
| `position` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'bottom'` | Side of the trigger where the tooltip appears               |
| `align`    | `'start' \| 'center' \| 'end'`           | `'center'` | Alignment along the cross-axis (ignored for `left`/`right`) |

## Accessibility note

The tooltip is purely CSS-positioned and mouse-driven. It does not currently implement `aria-describedby` or keyboard focus handling. For keyboard-accessible tooltips wrap the trigger in a focusable element and manage `onFocus` / `onBlur` alongside `onMouseEnter` / `onMouseLeave`.
