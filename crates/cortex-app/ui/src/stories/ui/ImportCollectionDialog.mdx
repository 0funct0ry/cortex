# ImportCollectionDialog

A multi-step portal dialog that guides the user through importing a Cortex collection from a ZIP archive or YAML bundle. Supports conflict detection when a collection with the same name already exists in the active workspace.

## Steps

```
[Pick file] → [Preview] → (conflict?) → [Conflict resolution] → [Importing…] → done
```

| Step        | Description                                                                       |
| ----------- | --------------------------------------------------------------------------------- |
| `preview`   | File picker opens automatically; on success shows collection name + request count |
| `conflict`  | If a collection with the same name exists, asks the user to choose a destination  |
| `importing` | Extraction command is in flight; spinner shown                                    |

## IPC Commands

| Command                     | Step                                        |
| --------------------------- | ------------------------------------------- |
| `pick_file`                 | On open — triggers native file picker       |
| `preview_import_zip`        | After file is selected (ZIP format)         |
| `preview_import_bundle`     | After file is selected (YAML bundle format) |
| `extract_collection_zip`    | Confirm button — ZIP extraction             |
| `extract_collection_bundle` | Confirm button — YAML extraction            |

## Store Dependencies

| Store            | Fields read                                                                                     |
| ---------------- | ----------------------------------------------------------------------------------------------- |
| `uiStore`        | `importCollectionDialog.isOpen`, `importCollectionDialog.format`, `closeImportCollectionDialog` |
| `workspaceStore` | `activeWorkspacePath`, `loadWorkspace`                                                          |

## Accessibility

- The dialog is a modal: backdrop click closes it.
- File selection errors are surfaced inline with a descriptive message.
- The Confirm button is disabled while the extraction command is in flight.
