# Theme Gallery

Cortex ships **13 themes** driven entirely by CSS custom properties. Every component uses tokens — never hardcoded colours — so switching themes requires only changing `data-theme` on the root `<html>` element. Use the **Theme** toolbar button at the top of this page to switch themes live.

The gallery below renders all 13 themes simultaneously. Each tile applies its own `data-theme` attribute so it is a fully independent themed island. A blank or incorrectly coloured tile immediately signals a missing or incomplete theme CSS file.

## How theming works

Themes are applied by setting the `data-theme` attribute on the `<html>` element. Because the selectors are element-agnostic (`[data-theme='light']`, not `:root[data-theme]`), you can also scope a theme to any subtree — which is how the gallery above renders all 13 themes simultaneously:

```tsx
// Each tile is a self-contained themed island
function ThemeTile({ id }: { id: string }) {
  return (
    <div data-theme={id} style={{ backgroundColor: 'var(--color-bg-base)' }}>
      {/* All CSS tokens resolve to this theme's values inside this div */}
    </div>
  )
}
```

## Adding a new theme

Three files must be updated when adding a new theme. The ThemeGallery story acts as a contract enforcement point — a blank tile means a step was missed.

**1. Create the CSS file**

```bash
# Copy the dark theme as a starting point
cp src/styles/themes/dark.css src/styles/themes/<new-id>.css
```

Edit `src/styles/themes/<new-id>.css` — change the selector and redefine all 39 tokens.

**2. Register in `ThemeContext.tsx`**

```ts
// src/contexts/ThemeContext.tsx
export type ThemeId =
  | 'dark' | 'light' | ...
  | '<new-id>'          // ← add here

export const THEMES: ThemeId[] = [
  'dark', 'light', ...,
  '<new-id>',           // ← add here
]
```

**3. Register in `.storybook/preview.tsx`**

```ts
withThemeByDataAttribute({
  themes: {
    dark: 'dark',
    ...
    '<new-id>': '<new-id>',   // ← add here
  },
  defaultTheme: 'dark',
  attributeName: 'data-theme',
})
```

After these three changes, the new theme tile will appear in the gallery above and in the toolbar dropdown.
