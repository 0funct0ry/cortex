# WelcomeScreen

The landing view shown when no workspace is currently loaded. It presents the Cortex logo, two primary action buttons (Create / Open workspace), and an optional list of recently opened workspaces.

The "Recent Workspaces" section is rendered only when `recentWorkspaces.length > 0`.

## Store Dependencies

This component accepts no props ‚Äî all state comes from `useWorkspaceStore`.

| Store            | Fields read / actions called                         |
| ---------------- | ---------------------------------------------------- |
| `workspaceStore` | `recentWorkspaces`, `loadWorkspace`, `openWorkspace` |

In Storybook stories, the store is pre-seeded in `beforeEach` via `useWorkspaceStore.setState`, and `loadWorkspace` / `openWorkspace` are replaced with `fn()` spies so interactions appear in the Actions panel.

## Primary Actions

| Button               | Action                                                                 |
| -------------------- | ---------------------------------------------------------------------- |
| **Create workspace** | Opens `CreateWorkspaceModal` (local React state ‚Äî no IPC on open)      |
| **Open workspace**   | Calls `openWorkspace()` ‚Üí triggers a native folder-picker dialog (IPC) |

## Recent Workspace Entry

Each recent workspace row shows the workspace **name** and its full **path** in monospace. Clicking a row calls `loadWorkspace(path)` to reopen it. On hover, an external-link icon appears.
